Gulp Sass Test

* Before running any Gulp tasks, you will need to install the command line tool globally: npm install --global gulp-cli *

To compile the Sass files into CSS, you will need to run the Gulp script, styles/gulp.js.

  • The dev task runs, compiles the .scss files and builds a non-minified CSS file with sourcemaps. We use this in conjunction with the watch task for developing.
  • To use the dev then watch task, run 'gulp' inside of the styles directory in your terminal. The dev task will build, then the watch task will watch your Sass files for any changes (saves)
  • The build task runs once and compiles the .scss. This builds with no sourcemaps and adds vendor prefixes
  • To build, run 'gulp build' inside of the styles directory.
